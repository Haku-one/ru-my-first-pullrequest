Интеграция с CDEK для WooCommerce с упрощенной формой адреса доставки.

== Description ==

CDEK Shipping for WooCommerce - это плагин для интеграции с службой доставки CDEK. Плагин предоставляет:

* Автоматический расчет стоимости доставки через API CDEK
* Выбор пунктов выдачи на основе города из адреса
* Упрощенную форму адреса (скрывает ненужные поля)
* Поддержку как pickup, так и courier доставки
* Интеграцию с API Яндекс.Карт для геокодирования

== Особенности ==

* Убирает поля "Населённый пункт", "Область/район", "Почтовый индекс" из формы
* Автоматически определяет город из поля "Адрес"
* Показывает доступные пункты выдачи CDEK для выбранного города
* Расчет стоимости доставки в реальном времени
* Поддержка бесплатной доставки при определенной сумме заказа
* Возможность добавления наценки к стоимости доставки

== Installation ==

1. Загрузите папку плагина в `/wp-content/plugins/`
2. Активируйте плагин через меню 'Плагины' в WordPress
3. Перейдите в настройки CDEK Shipping в меню "Настройки"
4. Введите ваши данные для API CDEK
5. Настройте зоны доставки в WooCommerce
6. Добавьте метод доставки "CDEK Shipping" в зону для России

== Configuration ==

= API Настройки =
* Account ID: Ваш идентификатор аккаунта CDEK
* Password: Пароль для API CDEK  
* Yandex Maps API Key: Ключ API Яндекс.Карт
* Test Mode: Использовать тестовую среду CDEK

= Настройка зон доставки =
1. Перейдите в WooCommerce > Настройки > Доставка
2. Создайте зону доставки для России
3. Добавьте метод доставки "CDEK Shipping"
4. Настройте параметры метода:
   - Город отправки
   - Наценка (%)
   - Сумма для бесплатной доставки
   - Типы доставки (пункты выдачи/курьер)

== Frequently Asked Questions ==

= Как настроить API CDEK? =
Получите данные для API в личном кабинете CDEK и введите их в настройках плагина.

= Почему не показываются пункты выдачи? =
Убедитесь, что в поле "Адрес" указан город, и что API CDEK настроен корректно.

= Как скрыть определенные поля? =
Плагин автоматически скрывает поля "Населённый пункт", "Область/район", "Почтовый индекс".
